# 早安日报采集方案

> 整理日期：2026-03-15
> 状态：✅ 测试成功

---

## 概述

早安日报是一个自动采集全球新闻资讯（政治、军事、经济、AI大模型）的脚本，每日早上自动运行并生成 JSON 数据文件。

---

## 脚本位置

```
/home/kaworu/.openclaw/workspace-taizi/scripts/fetch_morning_news.py
```

---

## 功能特点

1. **RSS 源采集**：从多个 RSS 源自动抓取新闻
2. **分类筛选**：自动将新闻分类到政治、军事、经济、AI大模型
3. **关键词过滤**：对军事和AI分类进行关键词匹配
4. **去重处理**：自动去除重复新闻
5. **用户加权**：支持自定义关键词加权排序
6. **自定义源**：支持添加自定义 RSS 源
7. **幂等锁**：防止重复采集

---

## RSS 源配置

| 分类 | RSS 源 |
|------|--------|
| 政治 | BBC World, Reuters, AP Top News |
| 军事 | Defense News, BBC World, Reuters |
| 经济 | Reuters Business, BBC Business, CNBC |
| AI大模型 | Hacker News, VentureBeat AI, MIT Tech Review |

---

## 输出文件

- 每日数据：`data/morning_brief_YYYYMMDD.json`
- 最新数据：`data/morning_brief.json`（看板读取用）
- 锁文件：`data/morning_brief_YYYYMMDD.lock`

---

## 使用方法

### 手动运行

```bash
cd /home/kaworu/.openclaw/workspace-taizi
python3 scripts/fetch_morning_news.py
```

### 强制重新采集

```bash
python3 scripts/fetch_morning_news.py --force
```

---

## Cron 配置（每日 06:00 自动采集）

在 OpenClaw cron 中配置：

```json
{
  "id": "早报采集",
  "agentId": "taizi",
  "name": "AI资讯简报",
  "enabled": true,
  "schedule": {
    "expr": "0 6 * * *",
    "kind": "cron",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "运行 fetch_morning_news.py 采集早报"
  }
}
```

## 进阶配置

### 添加自定义 RSS 源

编辑 `data/morning_brief_config.json`：

```json
{
  "categories": [
    { "name": "政治", "enabled": true },
    { "name": "军事", "enabled": true },
    { "name": "经济", "enabled": true },
    { "name": "AI大模型", "enabled": true }
  ],
  "keywords": ["AI", "大模型", "GPT", "Claude"],
  "custom_feeds": [
    {
      "name": "自定义源",
      "url": "https://example.com/rss.xml",
      "category": "科技"
    }
  ]
}
```
